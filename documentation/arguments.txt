Accepted command line arguments:

--s : does not save statistics on exit

--t : disables the use of the wordtree. Using this option enables the use of backspace in the typescrees

--p : does not save progress on exit

-f : loads file named f

-g [filename] : generates a text based on the Errortree

-a : run an analysis on the Errortree. Saves the results in a folder named letterMistypes.stat
	The results are in a gnuplot compatible format.
